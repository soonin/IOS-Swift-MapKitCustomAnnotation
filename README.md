# IOS-Swift-MapKitCustomAnnotation
``` Level : Intermediate ```

This tutorial learn you How to Create Custom Annotation Pin MapView with Swift ( MapKit , MKAnnotation, CLLocationCoordinate2D , MKMapView). This tutorial is made with Xcode 10 and built for iOS 12.

## About

Sample app to Customize Annotation Pin MapView ( MapKit , MKAnnotationView )

## Requirements

- iOS 12.0+
- Xcode 10.0+
- Swift 4.0+

## Usage

You won't need to Download any extra libaray just ```import UIKit``` inside the project
